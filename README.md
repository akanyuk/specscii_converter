## SpecSCII converter

Converting SpecSCII art exported from [zxart.ee/specscii](https://zxart.ee/specscii/) into trd and tap files

### Requirements

* [sjasmplus](https://github.com/z00m128/sjasmplus)

### Usage

**Windows:**
```
build.cmd basic_file
```

**Linux:**
```
./build.sh basic_file
```
